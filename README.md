# Global Azure Columbus 2026

Static conference website for Global Azure Columbus 2026 (April 18, 2026).

## What's in this repo

This monorepo contains the conference website built with [Astro](https://astro.build), located in the `cbus-global-azure-2026/` directory. The site uses Markdown content collections for event information and scheduling.

## Get started locally

1. Navigate to the site directory:
   ```bash
   cd cbus-global-azure-2026
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

The site will be available at `http://localhost:3000`.
